# Designing-and-implementing-a-SDN-routing-platform
Designing and implementing an SDN platform for centralized, flexible, and programmable network management. Focused on traffic engineering, automatic traffic classification with machine learning, and practical SDN architecture implementation with controllers, flow tables, and dynamic routing.
# Features
- Centralized and programmable network management
- Traffic engineering for optimized flow allocation
- Automatic traffic classification using supervised and unsupervised learning
- Implementation of SDN architecture including controllers and flow tables
- Dynamic routing based on network state

# Technologies Used
- **SDN Controllers:** OpenDaylight, Ryu
- **Protocols:** OpenFlow
- **Machine Learning:** Random Forest, Logistic Regression, K-Means
- **Simulation/Testing Environment:** Mininet / EVE-NG
